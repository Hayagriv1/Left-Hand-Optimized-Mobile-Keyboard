## Left-Hand Optimized Mobile Keyboard

This project focuses on designing an ergonomic mobile keyboard optimized for one-handed (left-hand) usage.

### Motivation
Traditional QWERTY layouts are not designed for thumb based typing on large smartphones, leading to increased thumb strain and inefficient reach.

### Approach
- Used Fitts’ Law to optimize key placement
- Incorporated anthropometric thumb data
- Designed a curved arc-based keyboard layout aligned with natural thumb motion
- Iteratively refined layouts based on usability and feedback

### Validation
- EMG sensors used to measure muscle activity
- Compared standard QWERTY vs proposed layout
- Achieved ~40–60% reduction in muscle activation and improved fatigue behavior

### Features
- Curved QWERTY layout
- Adjustable keyboard position and size
- Left/right hand adaptability

### Outcome
A balance between ergonomic efficiency and user familiarity, improving comfort without a steep learning curve.
